# Nomos (Greek for "Law") System Files

These are my various system files that drive system configuration(s).

## Install

### 1. Clone:

```bash
git clone https://github.com/jgttech/nomos.git ${HOME}/.nomos
```

### 2. Install:

```bash
~/.nomos/install.sh
source ${HOME}/.bashrc
```

### 3. Follow instructions in terminal.
