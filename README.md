# Nomos (Greek for "Law") System Files

These are my various system files that drive system configuration(s).

## Install

### 1. Clone:

```bash
git clone https://github.com/jgttech/nomos.git ~/.nomos
```

### 2. Install:

```bash
~/.nomos/install.sh
```

### 2. Enable:

```bash
~/.nomos/enable.sh
```
